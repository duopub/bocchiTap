**V1.0：**初版。

### 如何使用

这是一个可交互的网页 demo，以**解压**为目的制作了该网页，网页会随机生成 10 张图片，点击图片进行倍速播放，再次点击恢复初始的速度，点击刷新图标可以进行图片的刷新。（推荐使用谷歌浏览器）
